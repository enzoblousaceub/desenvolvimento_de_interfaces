Exemplo 22 CSS - Tipos de Vinculação de arquivos CSS

Existem 3 tipos básicos de vinculação de arquivos CSS, são elas: as vinculações externas e as internas que se dividem em duas incorpodas e inline.

Neste exemplo, nós estamos utilizando a vinculação externa, que é a mais indicada para projetos maiores

Para vincular arquivos CSS de forma externa, nós usamos a tag link.

Cores e estilos de Fundo

Uso do background com sistemas de cores RGB e RGBA

Sistema de cores RGB

Os sistemas de cores RGB representa um escquema de cores baseado nas cores R (red) G (green) B (blue), cada parammetro define a intensidade da cor e ela varia de 0 255. Por exemplo rgb(255,99,71)

Sistema de cores RGBA

O sitema de cores rgba são uma extensao dos valores rgb com um canal alfa (a), que especifica a opacidade que pode variar de 0 a 1, Sendo 0 totalmente transparente e o 1 totalmente opaco. Por exemplo rgba(35,99,71,0.3)
