Visao Geral sobre CSS

CSS é um acrônimo para folhas de estilo em cascata. Esta técnica visa personalizar, estilizar um documento HTML. Para isso, usamos regras CSS que representam uma estrutura que define como os elementos selecionados devem ser estilizados. Essa regra é composta pelas seguintes partes:

  • SeletorIndica quais elementos serão afetados pelo estilo definido.
  • Bloco de DeclaraçãoÉ o bloco composto por proprioedades e valores, esse bloco será aplicado ao elemento html.
